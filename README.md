## 🔧 Как добавить информацию

### Шаг 1️⃣ Отправьте данные

Поделитесь информацией о компоненте (сервер, сервис, сеть):

```
Сервер PROD-DB-01:
- ОС: Windows Server 2022
- IP: 192.168.1.10
- Роли: MSSQL, 1C
- Памяти: 128GB
- Ответственный: admin@example.com
```

### Шаг 2️⃣ Получите уточняющие вопросы

Если данных недостаточно, я спрошу детали:

| Поле | Вопрос |
|------|--------|
| `version` | Какая точная версия MSSQL (2019, 2022, какой CU)? |
| `backup.method` | Как создаются бэкапы (Veeam, SQL Agent, скрипт)? |

### Шаг 3️⃣ Подтвердите изменения

После получения всех данных я покажу вам результат:

```diff
+ title: "PROD-DB-01: MSSQL Primary Server"
+ version: "2022 CU5"
+ backup.method: "Veeam Backup & Replication"
```

**Подтвердить сохранение?**

### Шаг 4️⃣ Автоматическое логирование

Все изменения записываются в `.kb_logs/changes.log` с датой, автором и обоснованием.
