## Технические термины

### 1C Enterprise
- **Платформа:** Базовая система для разработки приложений
- **Кластер:** Группа 1C серверов приложений для распределения нагрузки
- **Сеанс:** Подключение пользователя к 1C приложению
- **Блокировка:** Механизм контроля конкурентного доступа

### MSSQL
- **Инстанс:** Отдельный экземпляр SQL Server на хосте
- **База данных:** Логический контейнер данных
- **Таблица:** Структурированный набор данных
- **Транзакция:** Атомарная операция (ACID)
- **Индекс:** Структура для ускорения поиска
- **Backup:** Копия базы данных на определённый момент времени
- **Восстановление (Restore):** Процесс восстановления данных из backup'а

### Сеть
- **IP адрес:** Уникальный идентификатор в сети
- **VLAN:** Виртуальное разделение физической сети
- **Subnet / CIDR:** Подсеть в нотации (например, 192.168.1.0/24)
- **Gateway:** Точка входа в другую сеть
- **Firewall:** Фильтр входящего/исходящего трафика

### Мониторинг
- **Метрика:** Измеряемое значение (CPU %, RAM, Disk I/O)
- **Алерт:** Уведомление при превышении порога
- **Dashboard:** Визуальное представление метрик
- **Log:** Запись о событии в системе

## Сокращения документов

- **KB** = Knowledge Base (База знаний)
- **TOC** = Table of Contents (Оглавление)
- **TBD** = To Be Determined (Требуется определить)
- **PENDING** = Требуется уточнение
- **NEEDS_VERIFICATION** = Требуется проверка
- **NEEDS_DATA** = Требуется информация от администратора
